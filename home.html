ABOUT ABTeC

ABTeC is the acronym of "Annotated Bangla Text Corpus". It is a system of concepts, structure and tools for the computer assisted transcription and annotation of Bangla written language, and for the construction and analysis of written language corpus. ABTeC was originally developed in the project "Smart dictionary – an annotate lexicon of most-frequently-used Bangla words for learners learning Bangla language" at the Department of Computer Science and Engineeirng, at the Shahjalal University of Science and Technology, Bangladesh. All components of the ABTeC system are freely available to users outside the Shahjalal University.

Main features of ABTeC
  • Balanced Text Corpus - ABTeC is the multidimensional and balanced text corpus of large size.
  • XML based data formats - All ABTeC data are stored in XML files. The use of this W3C standard ensures flexible usability and long-term archivability of the data.
  • Java based tools - All software tools for creating and working with ABTeC data are JAVA applications. This makes them suitable for all currently used operating systems (Windows, Macintosh, Linux, Unix).
  • Annotation - The ABTeC generates a well-structured and annotated lexicons to provide important insights about the linguistic information.

Furthermore, EXMARaLDA data can be transformed into a number of widely used presentation formats (RTF, HTML, XML) for web-based or printed publication. Last but not least, EXMARaLDA supports several important transcription systems (HIAT, DIDA, GAT, CHAT) through a number of parameterised functions.
